# 实体瘤 CAR-T × HCC：分层 Reward 体系简版

> 版本：2026-06-29  
> 场景：GPC3 阳性肝细胞癌（HCC）实体瘤 CAR-T 内部研发闭环  
> 目标：用分层 reward 体系，从大量 CAR 设计中筛出 1–3 个值得进入 IND-enabling（IND 申报前关键验证阶段）的候选。

---

## 1. 核心结论

HCC CAR-T 的 reward 体系不应围绕“杀伤率最高”设计，而应围绕：

```text
有效杀伤 + 肿瘤选择性 + T 细胞健康 + TME 抵抗力 + 肝安全窗口 + 可制造性
```

推荐结构是 7 层漏斗：

```text
L0 计算/患者数据
  -> L1 CAR 结构设计
  -> L2 2D 功能与抗原密度
  -> L3 多 donor 制造与 T cell fitness
  -> L4 HCC-TME 压力测试
  -> L5 3D 类器官/肿瘤切片
  -> L6 肝安全与器官芯片
```

前端高通量，后端高保真。类器官和器官芯片当然比临床试验快，但相对 L1/L2 的板式筛选仍然低通量，所以更适合做后段去风险，不适合作为主筛选平台。

## 2. Reward（评价信号）主轴

| 主轴 | 问题 | 典型读数 |
|---|---|---|
| Efficacy（有效性） | 能不能杀 HCC？ | killing（杀伤）、serial killing（连续多轮杀伤）、抗原低表达模型活性 |
| Selectivity（选择性） | 是否优先杀肿瘤？ | tumor/normal window（肿瘤与正常细胞反应差距）、正常肝相关细胞 counter-screen（反向安全筛查） |
| Fitness（T 细胞健康状态） | CAR-T 产品是否健康？ | 扩增、记忆表型、耗竭标志、donor 稳定性 |
| TME competence（微环境适应力） | 在 HCC 微环境下是否还有效？ | 低氧、TGF-β、CAF/TAM（肿瘤支持性细胞）、基质屏障下的功能保持 |
| Safety（安全性） | 是否有不可接受风险？ | cytokine（细胞因子/炎症信号）、肝毒性、内皮/巨噬细胞激活 |
| Developability（可开发性） | 能不能做成产品？ | 制造成功率、批间一致性、构型复杂度、CMC（生产工艺和质量控制）风险 |

决策方式：**硬性门槛 + 通过门槛后的排序**。安全性、选择性和制造稳定性不应被平均分“抵消”。

### 2.1 关键术语速查

| 术语 | 白话解释 |
|---|---|
| CAR-T | 把 T 细胞改造成能识别肿瘤的免疫细胞产品。 |
| HCC | 肝细胞癌，是最常见的原发性肝癌类型。 |
| GPC3 | HCC 中常见的肿瘤相关靶点，CAR-T 可被设计成识别它。 |
| reward | 内部评价信号，用来判断候选设计好不好，不等于单一总分。 |
| IND-enabling | 进入 IND 申报前的关键验证阶段，通常要证明候选具备继续开发的有效性、安全性和可制造性基础。 |
| donor | 提供 T 细胞或血液样本的人。CAR-T 表现会因 donor 差异而变化。 |
| TME | 肿瘤微环境，包括肿瘤周围的免疫抑制、基质、血管、低氧等因素。 |
| killing | CAR-T 杀死肿瘤细胞的能力。 |
| activation | CAR-T 被靶点刺激后是否被激活，例如释放信号分子或表达激活标志物。 |
| cytokine | 细胞因子，是免疫细胞释放的信号分子；过高可能提示炎症或 CRS 风险。 |
| CRS | 细胞因子释放综合征，CAR-T 常见风险之一，本质是免疫激活过强造成的全身炎症。 |
| IFN-γ / TNF-α / IL-2 / IL-6 / GM-CSF | 常测的细胞因子组合，用来观察 CAR-T 激活强度和炎症风险。 |
| TGF-β / IL-10 | 肿瘤微环境中常见的免疫抑制信号，会削弱 T 细胞功能。 |
| PD-L1 | 肿瘤或周围细胞表达的免疫抑制分子，可能让 T 细胞“刹车”。 |
| AFP | 甲胎蛋白，HCC 常见临床标志物之一。 |
| E:T ratio | effector:target ratio，即 CAR-T 与肿瘤细胞的数量比例。 |
| serial killing / serial stimulation | 连续多轮遇到肿瘤后，CAR-T 是否还能继续杀伤和保持功能。 |
| tumor-normal window | 肿瘤细胞反应与正常细胞反应之间的差距；差距越大，选择性通常越好。 |
| counter-screen | 反向安全筛查，用正常细胞测试候选是否会误伤正常组织。 |
| binder | CAR 上负责识别靶点的结合模块。 |
| hinge / spacer | CAR 的“连接臂”，影响识别模块离细胞膜的距离和灵活性。 |
| transmembrane | CAR 固定在 T 细胞膜上的跨膜区域。 |
| CD28 / 4-1BB | CAR 内部常用的共刺激信号域，会影响杀伤速度、持久性和耗竭风险。 |
| tonic signaling | 没有遇到靶点时 CAR-T 也自发激活，通常是坏信号。 |
| armoring | 给 CAR-T 加“增强模块”，例如抗 TGF-β、改善迁移或抗耗竭。 |
| PDO | patient-derived organoid，患者来源类器官，用患者肿瘤样本培养出的 3D 模型。 |
| CAF / TAM | 癌相关成纤维细胞 / 肿瘤相关巨噬细胞，常帮助肿瘤形成免疫抑制环境。 |
| MDSC / Treg | 免疫抑制性细胞，会压制 CAR-T 功能。 |
| Kupffer cell | 肝脏驻留巨噬细胞，可能放大肝内炎症。 |
| endothelial cell | 内皮细胞，构成血管内壁；其激活可能提示血管炎症或屏障破坏。 |
| dose window | 剂量窗，即既有效又相对安全的剂量范围。 |
| CD4/CD8 | T 细胞主要亚群比例，会影响产品功能和持久性。 |
| Tscm/Tcm | 更偏“长期记忆”的 T 细胞状态，通常与更好的持久性有关。 |
| terminal effector | 终末效应 T 细胞，短期杀伤可能强，但持久性通常较差。 |
| PBMC | 外周血单个核细胞，常作为获取 T 细胞的起始材料。 |
| ECM | 细胞外基质，类似肿瘤组织里的物理支架，可能阻碍 CAR-T 进入肿瘤。 |
| CMC | chemistry, manufacturing and controls，即生产工艺、质量控制和放行标准。 |
| QC | 质量控制，用来确认样本、细胞产品或实验批次是否合格。 |
| LIMS / ELN | 实验数据管理系统 / 电子实验记录本。 |
| IRB / MTA / SOP | 伦理审查 / 样本或材料转移协议 / 标准操作流程。 |
| TCGA / ICGC / GTEx / HPA | 常用公开数据库，分别偏癌症基因组、国际癌症队列、正常组织表达和蛋白表达。 |
| scRNA / 空间组学 / IHC | 单细胞 RNA、空间位置表达、免疫组化；用于看靶点在哪些细胞和组织区域表达。 |
| 高内涵成像 | 一种自动化显微成像和分析方法，可同时看细胞数量、位置、形态和空间分布。 |
| 流式 | 流式细胞术，用来快速测大量细胞的类型、状态和标志物。 |
| GMP / GLP | 临床生产质量规范 / 非临床安全评价质量规范。本文成本估算不包括正式 GMP 生产和 GLP 毒理全套。 |

## 3. 分层体系总表

| 层级 | 作用 | 具体做什么 | 典型通量 | 单轮时间成本 | 成本量级 | 输出 |
|---|---|---|---:|---:|---:|---|
| L0 计算与患者数据 | 判断靶点和患者人群是否值得做 | 公开癌症/正常组织数据库、单细胞/空间组学、免疫组化（IHC）、公开临床数据 | 10^3–10^5 个假设；10^3–10^6 个样本/细胞 | 数天–2 周；若加病理复核约 2–6 周 | 10–50 万美元/年 | 靶点风险图谱、患者覆盖率、IHC cut-off 假设 |

| L2 2D 功能与抗原密度 | 建立杀伤、抗原阈值、细胞因子关系 | GPC3 高/中/低/阴性 HCC 模型、正常细胞安全筛查、细胞因子、连续杀伤 | 30–300 个候选/轮 | 1–2 周 | 50–200 万美元/年 | 杀伤/激活窗口、细胞因子负担 |
| L3 多 donor 与 T 细胞健康状态 | 判断跨 donor 稳定性和可制造性 | 6–20 个 donor、扩增、表型、耗竭、批间差异 | 5–30 个候选/轮 × 6–20 donor | 3–6 周 | 50–200 万美元/年 | 制造成功率、donor 变异、产品质量 |
| L4 HCC-TME 压力测试 | 判断离开理想培养皿后是否还能工作 | 低氧、酸性/乳酸、TGF-β、PD-L1、CAF/TAM 等免疫抑制细胞、基质屏障 | 5–20 个候选/轮 × 4–10 条件 | 2–4 周 | 50–200 万美元/年 | TME 功能保留分数、是否需要增强模块（armoring） |
| L5 3D 类器官/肿瘤切片 | 验证空间浸润和患者异质性 | 患者来源类器官（PDO）、肿瘤球、肿瘤切片、CAF/内皮/髓系共培养、空间成像 | 3–10 个候选/轮 × 5–30 样本 | 2–8 周；新建 PDO 可更久 | 30–300 万美元/年 | 空间杀伤、浸润深度、残余克隆 |
| L6 肝安全与器官芯片 | 对少数候选做 IND 前安全去风险 | 正常肝细胞、胆管、肝内皮、Kupffer/巨噬细胞、肝类器官、肿瘤-血管/肝芯片 | 1–5 个候选/轮 × 10–100 条件 | 4–12 周 | 50–400 万美元/轮 | 肝毒性窗口、内皮/巨噬细胞炎症、剂量窗 |

## 4. 类器官与器官芯片的关系

类器官和器官芯片不是互相替代的关系，而是两个保真维度不同的模型。

| 模型 | 更擅长回答 | 不擅长回答 | 在体系中的位置 |
|---|---|---|---|
| 类器官 / 肿瘤切片 | CAR-T 能不能进入 3D 肿瘤结构、杀伤患者来源肿瘤、覆盖患者间异质性、是否留下 GPC3 低表达/阴性残余克隆 | 血流灌注、血管内皮激活、肝窦结构、动态剂量暴露、器官间连接、长期肝功能损伤 | L5，适合中后段验证 3–10 个候选 |
| 器官芯片 | 在流动、屏障、血管/肝组织结构下，CAR-T 是否能迁移、是否激活内皮/巨噬细胞、是否造成肝毒性、剂量窗是否足够 | 大规模筛选、覆盖大量患者样本、快速比较几百个构型 | L6，适合对 1–5 个候选做高保真去风险 |

所以，有了类器官不等于不需要器官芯片。更准确的说法是：**类器官可以显著减少需要上芯片的候选数量，但不能完全替代芯片。**

什么时候类器官基本够用：

- 目标问题是患者来源肿瘤杀伤、空间浸润、残余克隆和患者异质性。
- 候选还在 discovery 阶段，尚未收敛到 IND 前候选。
- 公司预算有限，只需要先证明“细胞系里强”的候选在更接近患者肿瘤的模型里仍然有效。

什么时候仍然需要器官芯片：

- 需要评估血流/灌流条件下的 CAR-T 迁移和肿瘤进入能力。
- 需要评估肝窦内皮、Kupffer/巨噬细胞和胆管等正常肝相关组织的安全风险。
- 需要观察内皮炎症、屏障破坏、巨噬细胞放大细胞因子等机制。
- 需要做低/中/高暴露水平下的剂量窗和 IND 前 go/no-go 判断。

最实用的组合是：L5 用类器官/切片把 5–10 个候选缩到 2–5 个；L6 再用肝安全模型和器官芯片对 1–5 个候选做去风险。类器官负责“更像患者肿瘤”，器官芯片负责“更像器官环境和动态暴露”。

## 5. 湿实验与计算替代标注

这里的“计算替代”不应理解成完全不做实验，而是用计算模型减少湿实验数量、优先级排序、选择最有信息量的实验条件。对 CAR-T 候选物选择来说，L0 可以主要计算完成；L1 可以大量计算预筛；L2–L6 都需要关键湿实验确认。

| 层级 | 湿实验必要性 | 可计算替代/增强的部分 | 计算输入 | 计算输出 | 不能被计算替代的部分 |
|---|---|---|---|---|---|
| L0 计算与患者数据 | 可主要计算完成，少量病理/表达验证更好 | 靶点筛选、患者覆盖率、正常组织风险、分层假设 | 公开癌症队列、正常组织表达、单细胞/空间组学、IHC、临床队列 | 靶点风险图谱、患者覆盖率、GPC3 高/中/低分层、需要验证的正常组织清单 | 蛋白膜定位、IHC cut-off 的病理复核、真实样本质量差异 |
| L1 CAR 结构设计 | 需要湿实验确认，但可大量计算预筛 | binder 排序、亲和力区间、构型复杂度、潜在自发激活风险 | CAR 序列/结构、GPC3 结构或表位信息、历史表达/激活数据、载体长度和模块组合 | 推荐构型列表、亲和力优先级、tonic signaling 风险排序、进入 L2 的候选池 | CAR 表达、表面定位、primary T cell 中的自发激活、真实制造可行性 |
| L2 2D 功能与抗原密度 | 必须湿实验 | 实验矩阵设计、杀伤曲线预测、细胞因子风险预测、主动学习选点 | L1 构型特征、抗原密度、细胞系组学、历史 killing/cytokine 曲线、E:T ratio 和时间点 | 预测杀伤曲线、激活阈值、cytokine/killing 风险、最值得做的实验组合 | 实际 killing、activation、细胞因子释放、正常细胞安全筛查 |
| L3 多 donor 与 T 细胞健康状态 | 必须湿实验 | donor 失败风险、扩增/耗竭趋势、批次异常预警 | donor metadata、制造过程参数、转导率、扩增曲线、表型/耗竭标志、历史批次数据 | donor 鲁棒性预测、制造失败风险、批次质量预警、候选稳定性排序 | 多 donor 实际制造、扩增、产品表型、跨 donor 功能分布 |
| L4 HCC-TME 压力测试 | 必须湿实验 | 优先选择 TME 压力条件、预测失败机制、armoring 方向建议 | L2/L3 数据、HCC 单细胞/空间图谱、TGF-β/PD-L1/CAF/TAM 等 TME 特征、迁移/趋化因子数据 | TME 功能保留预测、最关键压力条件、潜在失败机制、增强模块建议 | 低氧、TGF-β、CAF/TAM、基质屏障下的真实功能保持 |
| L5 3D 类器官/肿瘤切片 | 必须湿实验 | 图像分析、空间杀伤定量、患者响应分群、残余克隆预测 | 类器官/切片图像、空间组学、GPC3 表达图、L2/L4 功能数据、患者样本特征 | 浸润深度、空间杀伤图、患者响应分布、非响应样本特征、逃逸克隆风险 | 患者来源模型中的真实浸润、杀伤、残余克隆和样本异质性 |
| L6 肝安全与器官芯片 | 必须湿实验 | 毒性机制整合、剂量窗建模、芯片条件选择、风险排序 | 正常肝细胞表达、L2 cytokine、L5 空间杀伤、肝安全/芯片读数、剂量暴露假设、历史毒性特征 | 肝毒性风险排序、内皮/巨噬细胞炎症风险、建议剂量窗、IND 前风险缓解策略 | 正常肝/胆管/内皮/Kupffer 反应、芯片灌流下的真实炎症和肝功能损伤 |

推荐的计算闭环是：先用 L0/L1 计算模型生成候选和实验优先级；用 L2/L3 湿实验产生第一批训练数据；再用 L4–L6 高保真湿实验校准早期 reward。真正有价值的不是“用计算替代所有实验”，而是让每一轮湿实验更少、更准、更能解释失败机制。

## 6. 每层关键 Reward 与淘汰信号

| 层级 | 关键 reward | 主要淘汰/降级信号 |
|---|---|---|
| L0 | 靶点覆盖率、抗原均一性、正常组织风险、患者分层可执行性 | 靶点覆盖太窄；正常组织风险明显；无法定义可执行入组标准 |
| L1 | CAR 表达质量、自发激活（tonic signaling）、结合特异性、结构简洁性 | 无抗原自发激活；表达不稳定；结构过复杂导致制造风险高 |
| L2 | 肿瘤杀伤、激活阈值、细胞因子负担、肿瘤-正常窗口、连续杀伤 | GPC3 阴性细胞被激活；正常细胞被杀伤；细胞因子高但杀伤收益低 |
| L3 | 制造成功率、扩增稳定性、产品表型质量、donor 间变异 | 多 donor 扩增失败；批间差异大；快速耗竭；终末效应 T 细胞比例偏高 |
| L4 | TME 功能保留、迁移能力、抗抑制能力、压力下耗竭、是否需要增强模块 | 低氧/TGF-β/CAF/TAM 下功能塌陷；为补救需要过多增强模块 |
| L5 | 空间浸润、空间杀伤、残余克隆、患者样本异质性 | 只能杀外层；多个患者样本无响应；残余 GPC3 低表达/阴性克隆富集 |
| L6 | 肝毒性窗口、胆管风险、内皮激活、巨噬细胞炎症放大、剂量窗 | 正常肝/胆管/内皮损伤；巨噬细胞放大 IL-6/TNF-α 等炎症信号；安全窗口过窄 |

## 7. 漏斗比例

| 阶段 | 输入 | 输出 |
|---|---:|---:|
| L0/L1 | 10^2–10^4 个构型/假设 | 30–100 个候选 |
| L2 | 30–100 个候选 | 10–30 个候选 |
| L3/L4 | 10–30 个候选 | 5–10 个候选 |
| L5 | 5–10 个候选 | 2–5 个候选 |
| L6 | 1–5 个候选 | 1–3 个 IND-enabling 候选 |

临床试验不应被当作筛选层。临床前 reward engine 的任务，是尽量把明显失败模式提前暴露出来。

## 11. 最小可行版本

资源有限时，第一版只需要搭下面 7 件事：

1. GPC3 表达梯度 HCC 模型。
2. 正常肝相关细胞安全筛查。
3. 多 donor CAR-T 制造和功能验证。
4. 2D 杀伤 + 细胞因子 + 连续刺激。
5. HCC-TME 压力测试：低氧、TGF-β、CAF/TAM 至少覆盖两到三类。
6. 患者来源类器官或肿瘤切片小规模验证。
7. 少数候选再上肝类器官、肝芯片或肿瘤-血管芯片。

第一版预算建议按 **300–600 万美元 / 12–18 个月** 规划。先证明低保真 L1/L2 reward 是否能预测高保真 L4/L5/L6 结果；如果相关性成立，再扩成稳定的内部平台。

## 12. 各层目标与执行方法展开

### L0：计算与患者数据

**目标**  
回答“这个靶点和患者人群值不值得做”。L0 不证明 CAR-T 有效，它定义边界：适用人群多大、正常组织风险在哪里、后续实验应该优先模拟哪些患者亚型。

**怎么做**  
整合公开癌症数据库、单细胞和空间组学、免疫组化（IHC）、正常组织表达数据库和公开临床资料，建立 GPC3 表达覆盖率、肿瘤内异质性、正常组织表达和可溶性抗原风险图谱。输出第一版患者分层规则，例如 GPC3 IHC cut-off（免疫组化阳性判定阈值）、GPC3 高/中/低表达模型定义，以及必须进入正常细胞安全筛查的组织列表。

### L1：CAR 结构设计

**目标**  
从大量 CAR 构型中快速排除表达差、自发激活高、结构复杂或制造风险大的设计。

**怎么做**  
围绕靶点结合模块（binder）、亲和力、连接臂（hinge/spacer）、跨膜锚定区（transmembrane）、共刺激信号域（CD28/4-1BB）和增强模块（armoring）预留位点建立小型到中型设计库。先测 CAR 表达、表面定位、无抗原自发激活（tonic signaling）和基础结合特异性，再把明显有风险的构型剔除。L1 的输出不是最终候选，而是进入 L2 的几十到上百个可测构型。

### L2：2D 功能与抗原密度

**目标**  
建立每个候选的基础功能曲线：在不同 GPC3 密度下能不能杀、需要多少抗原触发、杀伤是否伴随过高细胞因子。

**怎么做**  
构建 GPC3 高、中、低、阴性 HCC 细胞模型，并加入正常肝相关细胞和关键非肝正常细胞。用多个 CAR-T:肿瘤细胞比例（E:T ratio）和多个时间点，测杀伤、激活、IFN-γ/TNF-α/IL-2/IL-6/GM-CSF 等细胞因子，以及连续多轮杀伤能力。重点不是单点杀伤率，而是肿瘤-正常窗口、激活阈值和“细胞因子/杀伤收益”比例。

### L3：多 donor 与 T Cell Fitness（T 细胞健康状态）

**目标**  
判断候选是否在不同 donor 中稳定制造、稳定扩增，并保持可持续杀伤所需的 T 细胞状态。

**怎么做**  
用 6–20 个健康 donor 或疾病相关 donor 进行转导、扩增和功能验证。记录制造成功率、扩增倍数、CD4/CD8、Tscm/Tcm、耗竭标志、批间差异和 donor 间功能分布。L3 要看分布而不是平均值：一个候选平均杀伤强但 30% donor 失败，通常不如杀伤略低但跨 donor 稳定的候选。

### L4：HCC-TME 压力测试

**目标**  
判断候选离开理想 2D 培养条件后，在 HCC 免疫抑制微环境中是否还能工作，并识别是否需要增强模块。

**怎么做**  
设置低氧、酸性/乳酸、TGF-β、IL-10、PD-L1、高 AFP、可溶性 GPC3、CAF（癌相关成纤维细胞）、肝星状细胞、TAM/MDSC/Treg 等免疫抑制条件。比较压力条件下功能相对基础条件的保留比例，测迁移、耗竭、细胞因子和功能保留。L4 的关键输出是失败机制：例如对 TGF-β 敏感、迁移差、被 CAF/ECM（细胞外基质）阻隔、低氧下耗竭。

### L5：3D 类器官 / 肿瘤切片

**目标**  
验证候选是否能进入真实 3D 肿瘤结构，是否能处理患者间异质性和肿瘤内残余克隆。

**怎么做**  
使用患者来源 HCC 类器官、HCC 肿瘤球和新鲜肿瘤切片，优先加入 CAF、内皮或髓系免疫细胞共培养。用高内涵成像、空间成像或流式/测序读出浸润深度、肿瘤内部杀伤、残余 GPC3 低表达/阴性克隆和患者样本响应分布。L5 适合验证 5–10 个候选，不适合前端大规模筛选。

### L6：肝安全与器官芯片

**目标**  
对 1–5 个最优候选做高保真安全去风险，重点回答 HCC 患者脆弱肝脏中是否存在不可接受的肝毒性、胆管风险、内皮炎症或巨噬细胞放大效应。

**怎么做**  
先做正常肝细胞、胆管细胞、肝窦内皮、Kupffer/macrophage（肝脏巨噬细胞）和肝星状细胞的安全筛查，再用肝类器官/肝球模型观察 albumin（白蛋白）、urea（尿素）、CYP（药物代谢酶）和胆汁酸相关损伤。最后对少数候选进入肿瘤-血管或肿瘤-肝脏器官芯片，比较低/中/高暴露水平下的剂量窗、局部炎症、屏障破坏和肝功能损伤。L6 的输出应是 IND 前 go/no-go 和风险缓解策略，不应被包装成“预测临床成功”。
